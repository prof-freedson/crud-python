ATENÇÃO: primeiramente, vamos remover as credenciais
do Github no Windows, caso tenham sido criadas

1) Fluxo no Windows:

Menu Iniciar -> Gerenciador de Credenciais -> Credenciais do Windows

2) Selecione a credencial que contém 'github'

3) Clique em "Remover" e confirme em seguida.

======================================
Principais comandos básicos do Git
======================================

1) Configurando nome do usuário
git config --global user.name nome-do-seu-usuario

2) Configurando e-mail do usuário
git config --global user.email email-do-seu-usuario

3) Verificar o usuário configurado
git config user.name

4) Verificar o e-mail configurado
git config user.email

5) Iniciar um repositório
git init

6) Adicionar os arquivos para
o envio ao Github
git add .

7) Adicionar mensagem sobre
os arquivos para o envio 
ao Github
git commit -m "Sua mensagem"

8) Alterar a nome da ramificação
do repositório local (computador
ou notebook) para 'main'
git branch -M main

9) Referenciar o repositório local
para o repositório remoto (Github)
git remote add origin https://github.com/seu-repositorio-github.git

10) Realizar o envio do repositorio
local para o remoto
git push -u origin main

OBS: depois de confirmado o envio do repositório
local para o remoto, toda alteração posterior
deverá ser enviada ao Github com os comandos
a seguir

git add .

git commit -m "Nova mensagem"

git push
